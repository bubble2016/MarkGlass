# 更新日志

## 1.0.1 - 2026-06-19

### 修复

- 按下 `ESC` 可直接关闭阅读窗口。
- 同目录文件栏默认隐藏，折叠后不再残留或覆盖阅读卡片。
- 普通 Markdown 图片恢复为行内自适应布局，表格恢复为完整宽度。
- 文档目录改为右下角固定毛玻璃组件，滚动后仍可展开和快速跳转。
- 消除启动白屏，增加约 360ms 的 MarkGlass 品牌过渡。

### 优化

- 底部操作栏改为约 50% 透明的弱化毛玻璃样式，悬停时提高可见度。
- 演示页的图片、表格和 Mermaid 双栏布局仅作用于演示文档。

## 1.0.0 - 2026-06-19

MarkGlass 首个正式大版本。

### 新增

- 全屏沉浸式 Markdown 阅读界面，采用无边框桌面覆盖层布局。
- Windows Acrylic 毛玻璃背景与半透明侧栏、工具栏。
- 中央白色阅读卡片、文档切换、目录、缩放、主题和全屏控制。
- 同目录 Markdown 文件列表、阅读进度与文档缩略图。
- Mermaid、代码高亮、任务列表、表格和脚注等 Markdown 渲染能力。
- 打开与关闭阅读卡片的缩放过渡动画。

### 设计

- 全面更新 MarkGlass 品牌标识、字标和应用图标。
- 更新 Windows、macOS、iOS 与 Android 平台图标资源。
- 移除标题前的 Markdown `#` / `##` 视觉标记，使阅读界面更简洁。
- 增加亮色、深色和护眼主题。

### 工程

- 引入 Windows 窗口振动效果支持。
- 完成 Vue、TypeScript 与 Tauri 配置的正式版整理。
